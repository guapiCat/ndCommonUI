##### 属性配置

| 参数                    | 说明                       | 类型        | 默认值                                        |
| -------------------     | ------------------------  | ----------- | --------------------------------------------- |
| showArrow   |   为true表示带箭头的提示信息           | boolean       | true                     |
| content               |  提示语内容信息           | string     | --
| placement            |  提示语的位置信息      | string     | --
| effect            |  是否是深夜模式，为true则深夜模式，false为白色模式      | boolean     | true
